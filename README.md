# 🌬️ Smart Temperature & Gesture-Controlled Fan System

## 🚀 Overview  
A smart fan control system that combines **temperature sensing + gesture detection + manual control** to create an energy-efficient and user-aware cooling solution.

Unlike traditional systems, this setup ensures the fan runs **only when both environmental conditions and user intent are satisfied**.

---

## 🎯 Problem Statement  
Conventional fans:
- Run unnecessarily → waste energy  
- Require manual operation  
- Lack user interaction  

👉 This project solves that by introducing **context-aware automation** using sensors.

---

## ✨ Features  
- 🌡️ Temperature-based control using DHT11  
- ✋ Gesture detection using ultrasonic sensor (HC-SR04)  
- 🔘 Button-based system activation toggle  
- 💡 LED output (can be replaced with relay-controlled fan)  
- 🔌 Compatible with real-world appliances via relay  
- 🧼 Contactless operation after activation
